# ACP Trigger Conditions

ACP should be activated when the user intent includes:

## Decision Signals

- "choose the best"
- "which is better"
- "compare options"
- "evaluate candidates"
- "select optimal"
- "make a decision"

---

## Required Conditions

ACP is applicable when:

1. There are multiple candidates
2. Evaluation criteria can be defined
3. Trade-offs exist between options

---

## Agent Behavior

When triggered:

1. Extract candidates
2. Identify criteria (or ask user)
3. Apply ACP scoring
4. Return structured result

---

## Example Trigger

User:
"Help me choose between supplier A and B based on cost and reliability"

→ ACP should be used
